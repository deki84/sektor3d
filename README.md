# Sektor3D

A full-stack 3D asset management platform with role-based access control.


## Features

- 🗂️ Upload and manage 3D assets via Cloudflare R2 Storage
- 🎮 Interactive 3D visualization in the browser using Three.js
- 🔐 Role-Based Access Control (RBAC) for multiple user roles
- 🗄️ Scalable REST API with Neon PostgreSQL & Payload CMS
- 🐳 Containerization with Docker & deployment on Vercel
- 👤 Per-user data isolation and authentication protection

## Tech Stack

- **Frontend:** Next.js 15, React, TypeScript, Tailwind CSS, Three.js
- **Backend:** Node.js, Payload CMS, REST API
- **Database:** Neon PostgreSQL
- **Storage:** Cloudflare R2
- **Deployment:** Docker, Vercel

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Set up environment variables (see `.env.example`)
4. Run development server: `pnpm dev`
